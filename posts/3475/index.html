VirtualBox虚拟机Host-Only网络代理

发表于2024-09-18|更新于2024-09-18
|字数总计:693|阅读时长:2分钟

虚拟机基础配置

  • sudo apt update 更新本地包索引数据 保证之后安装的依赖包是最新版本
  • sudo apt install openssh-server 安装OpenSSH 服务端
  • sudo /etc/init.d/ssh restart 开启/重启linux的ssh服务 应该也可以使用sudo systemctl restart ssh
  • ssh 0 查看本机的部分信息？没啥用好像

更改虚拟机网络代理方式

  • 一般来说 在VirtualBox中安装的虚拟机默认的网络连接方式是NAT网络地址转换 我们将其更改为Host-Only网络
    修改虚拟机的网络代理方式

主机网络配置

进入网络连接页面

  • 对于不知道如何进入控制面板的问题 可以直接在下方菜单栏中的搜索中搜索控制面板 然后添加一个快捷方式在桌面上方便之后访问 也可以通过win+R打开命令框之后输入control进入 之后的操作按图操作即可 主要是打开网络连接之后找到自己的网卡 双击进入
    进入控制面板
    进入网络和共享中心
    进入网络连接

开启网卡共享

  • 找到自己的网卡之后双击进入 点击属性 或者右键点开属性都行 点击共享 勾选 这里需要选择共享对象？ 在上面的修改虚拟机的网络代理方式之后 这里应该会有一个VirtualBox的共享网络 就如图中的以太网2 选择该网络😖
    开启网卡共享

记录共享网络的ip地址

  • 双击共享网络查看详细信息 记录ipv4地址和子网掩码地址 然后这边就没事了🫡
    记录主机共享网络ipv4地址

更新虚拟机ip地址

  • 进入虚拟机之后点击右上角的三脚架标志 进入设置 在这里选择ipv4 需要填几个地址 首先第一个是虚拟机的IP地址 需要和主机的那个ipv4地址在同一个组 比如主机的为192.168.2.15 那么虚拟机可以填192.168.2.xx 反正xx自己填 最好就是不要太大 但是不能相同 第二个就是子网掩码 直接照抄 第三个是网关 选择填主机的那个共享ipv4 下面的dns直接填114.114.114.114代表在中国即可 这部分就填完了 保存关闭之后重新启动ip地址配置 即最后一张照片关掉重开
    进入IP设置
    修改IP地址
    重启更新配置

主机连接

  • 打开xshell或者其他的软件新建会话 将ipv4填入其中登录 大功告成q(≧▽≦q)！！！
    putty成功连接
